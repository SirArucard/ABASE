# 🎟️ ABase - Sistema de Eventos

Sistema web para descoberta e gerenciamento de eventos próximos.

## 🚀 Funcionalidades
- 📍 Busca por localização
- 🗺️ Mapa interativo
- 👤 Perfil de usuário
- 🎟️ Compra e check-in
- 🏗️ Criação de eventos

## 🛠️ Tecnologias
- PHP
- MySQL
- JavaScript
- Leaflet
